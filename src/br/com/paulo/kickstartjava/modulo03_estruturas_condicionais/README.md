## Módulo 03 – Estruturas Condicionais

Conteúdos estudados neste módulo:

- Estruturas condicionais simples
- Estruturas condicionais compostas (teórica)
- Operadores lógicos AND, OR e NOT
- Switch Case
- Operador ternário

Cada pacote representa uma aula do curso, respeitando a organização didática original.
