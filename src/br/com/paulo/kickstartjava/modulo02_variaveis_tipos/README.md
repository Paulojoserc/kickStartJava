## Módulo 02 – Variáveis e Tipos de Dados

Conteúdos estudados neste módulo:

- Declaração e uso de variáveis
- Tipos primitivos
- Números inteiros e ponto flutuante
- Booleanos
- Textos (String)
- Conversão de tipos
- Operadores aritméticos

Cada pacote representa uma aula do curso, mantendo a organização didática original.
